# nivant-docmind

Chat with your documents using RAG — upload PDFs, ask questions, get AI-powered answers

## Tech Stack

React, TypeScript, Vite, Tailwind CSS, Shadcn UI

## Quick Start

```bash
git clone https://github.com/engineering-nivantlabs/nivant-docmind.git
cd nivant-docmind
npm install
cp .env.example .env
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Environment Variables

Copy `.env.example` to `.env` and fill in your keys:

```bash
cp .env.example .env
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Configured for Vercel — push to `main` and it auto-deploys.
